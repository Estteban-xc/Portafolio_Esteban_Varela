ERP NovaTek

Sistema ERP Modular · Empresas Colombianas

Sistema de planificación de recursos empresariales (ERP) modular desarrollado para pequeñas y medianas empresas colombianas. Cada módulo funciona de forma completamente independiente, compuesto por cinco módulos: Inventario, Recursos Humanos, Ventas y Facturación Electrónica, CRM y Contabilidad. Cumple con normativa colombiana vigente, incluyendo facturación electrónica DIAN.

Tecnologías

C# .NET SQL Server ADO.NET Visual Studio 2022 WPF / WinForms PlantUML
Arquitectura: Capas — Presentación / Lógica de Negocio / Acceso a Datos

DOOM'nt

DOOM'nt

Videojuego FPS · Raycasting 3D

Videojuego de disparos en primera persona (FPS) inspirado en el clásico DOOM, desarrollado en Python. El jugador se mueve por un mapa 3D generado con raycasting y debe eliminar a todos los enemigos. Incluye tres tipos de enemigos con IA (patrullaje, persecución, ataque), sistema de salud con recuperación automática, efectos de sonido, música de fondo y pantalla de victoria.

Tecnologías

Python 3 Pygame Raycasting BFS / lru_cache PyInstaller
Distribución: Ejecutable .exe para Windows vía PyInstaller

STRNG

STRNG

Sitio Web Estático · Fitness & Salud

Sitio web estático sobre entrenamiento físico, nutrición y salud con estética oscura y agresiva tipo gymrat. Siete páginas: hero con partículas animadas, rutinas, alimentación y suplementación, catálogo de implementos, calculadora de IMC, contacto. Incluye modo oscuro/claro, menú responsive hamburguesa, ticker animado, música de fondo y video embebido.

Tecnologías

HTML5 CSS3 JavaScript Vanilla Google Fonts Canvas API
Nota: Sin frameworks externos — 100% HTML, CSS y JS nativo
